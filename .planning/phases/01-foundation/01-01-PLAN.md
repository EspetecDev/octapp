**Step 3 — Write `svelte.config.js`:**

```javascript
import adapter from "@sveltejs/adapter-static";
import { vitePreprocess } from "@sveltejs/kit/vite";

/** @type {import('@sveltejs/kit').Config} */
const config = {
  preprocess: vitePreprocess(),
  kit: {
    adapter: adapter({
      pages: "build",
      assets: "build",
      fallback: "index.html",
      precompress: false,
      strict: false,
    }),
  },
};

export default config;
```

The `fallback: "index.html"` is critical — it ensures SPA routing works when the Bun server falls back to `index.html` for unknown paths (Pitfall 5 from RESEARCH.md).

**Step 4 — Write `vite.config.ts`:**

```typescript

export default defineConfig({

});
```

NOTE: `tailwindcss()` must be listed BEFORE `sveltekit()` in the plugins array.

```css
@theme {

  --color-bg: #0f0f0f;
  --color-surface: #1c1c1e;
  --color-border: #2d2d2f;

  /* Text */
  --color-text-primary: #f9fafb;
  --color-text-secondary: #9ca3af;

  /* Role accents */
  --color-accent-groom: #f59e0b;
  --color-accent-group: #ef4444;
  --color-accent-admin: #6b7280;

  /* Semantic */
  --color-success: #22c55e;
  --color-destructive: #ef4444;

  /* Typography */
  --font-size-label: 14px;
  --font-size-body: 16px;
  --font-size-heading: 24px;
  --font-size-display: 40px;

  /* Spacing (multiples of 4) */
  --spacing-xs: 4px;
  --spacing-sm: 8px;
  --spacing-md: 16px;
  --spacing-lg: 24px;
  --spacing-xl: 32px;
  --spacing-2xl: 48px;
  --spacing-3xl: 64px;

  /* Animation durations */
  --duration-fast: 100ms;
  --duration-normal: 200ms;
  --duration-slow: 300ms;
}

/* Global reset for mobile — no scroll bounce on body (RESEARCH.md Interaction Constraints) */
html,
body {
  margin: 0;
  padding: 0;
  background-color: #0f0f0f;
  color: #f9fafb;
  font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;
  font-size: 16px;
  overscroll-behavior: none;
  -webkit-tap-highlight-color: transparent;
}

/* Touch-friendly tap delay elimination on all interactive elements */
button,
a,
input,
select,
[role="button"] {
  touch-action: manipulation;
}

/* Focus ring — pre-role-selection default (role-specific override set via CSS var at runtime) */
:focus-visible {
  outline: 2px solid #f9fafb;
  outline-offset: 2px;
}

/* Minimum tap target enforcement */
button,
[role="button"],
input[type="submit"] {
  min-height: 44px;
  min-width: 44px;
}
```

**Step 1 — Write `Dockerfile`:**

The build sequence is: install deps → build SvelteKit static bundle → run Bun server.

```dockerfile
FROM oven/bun:1 AS builder

WORKDIR /app

# Copy package files first for layer caching
COPY package.json bun.lockb* ./

# Install ALL dependencies (including devDependencies needed for build)
RUN bun install --frozen-lockfile

# Copy source
COPY . .

# Build SvelteKit static bundle into /app/build
RUN bun run build

# Production image — smaller, only runtime needed
FROM oven/bun:1-slim AS runner

WORKDIR /app

# Copy the built static files and server source
COPY --from=builder /app/build ./build
COPY --from=builder /app/server ./server
COPY --from=builder /app/package.json ./package.json

# Install only production dependencies (server has none currently, but future-proof)
RUN bun install --production --frozen-lockfile 2>/dev/null || bun install --production

# Railway injects PORT at runtime; default 3000 for local
ENV PORT=3000

EXPOSE $PORT

# Entry point: Bun runs the WebSocket + static file server
CMD ["bun", "run", "server/index.ts"]
```

**Step 2 — Write `railway.toml`:**

```toml
[build]
builder = "dockerfile"
dockerfilePath = "Dockerfile"

[deploy]
healthcheckPath = "/health"
healthcheckTimeout = 10
restartPolicyType = "on-failure"
restartPolicyMaxRetries = 3
```

NOTE: The `/health` endpoint must be implemented in the Bun server (Plan 02) — it returns `200 OK`.
