The component must implement:

```svelte
<script lang="ts">
  let { duration, onExpire }: { duration: number; onExpire: () => void } = $props();
  let remaining = $state(duration);
  const RADIUS = 45;
  const CIRCUMFERENCE = 2 * Math.PI * RADIUS;

  $effect(() => {
    const interval = setInterval(() => {
      remaining -= 1;
      if (remaining <= 0) {
        clearInterval(interval);
        onExpire();
      }
    }, 1000);
    return () => clearInterval(interval);
  });

  let strokeColor = $derived(
    remaining > duration * 0.5 ? "#22c55e"
    : remaining > duration * 0.25 ? "#f59e0b"
    : "#ef4444"
  );
  let dashOffset = $derived(CIRCUMFERENCE * (1 - remaining / duration));
</script>

<svg viewBox="0 0 100 100" class="w-24 h-24" role="timer" aria-label="{remaining} seconds remaining">
  <circle cx="50" cy="50" r={RADIUS} fill="none" stroke="#2d2d2f" stroke-width="8"/>
  <circle
    cx="50" cy="50" r={RADIUS}
    fill="none"
    stroke={strokeColor}
    stroke-width="8"
    stroke-linecap="round"
    stroke-dasharray={CIRCUMFERENCE}
    stroke-dashoffset={dashOffset}
    style="transform: rotate(-90deg); transform-origin: 50% 50%; transition: stroke-dashoffset 1s linear, stroke 300ms ease;"
  />
  <text x="50" y="56" text-anchor="middle" fill="#f9fafb" font-size="24" font-weight="700">{remaining}</text>
</svg>
```

Notes:
- 96px × 96px (`w-24 h-24` = 96px in Tailwind)
- Track circle: stroke `#2d2d2f` (--color-border)
- Progress circle: stroke color derived from remaining time thresholds (>50% green, 25-50% amber, <25% red)
- CSS transition on stroke-dashoffset (1s linear) and stroke color (300ms ease)
- `role="timer"` and `aria-label` for accessibility (UI-SPEC)
- The `$effect` returns a cleanup function to `clearInterval` — prevents Pitfall 6 (timer continues after win)
